Comment compiler un fichier Java à partir de la ligne de commande?
------------------------------------------------------------------

Pour compiler un fichier source Java (File.java, dans cette example)
il suffit de taper:

javac File.java

Si la compilation se terminera correctement, cette operation produira
un fichier qui contient du bytecode Java (File.class),
qui pourra etre exceute grace a la commande:

java File
